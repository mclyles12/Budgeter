# Budgeter

A personal budget tracking web app with AI-powered chat assistant.

## Features

- **AI Chat Assistant**: Natural language interface for logging expenses, income, and savings
- **Transaction Ledger**: View and manage all financial transactions by month
- **Bills Overview**: Track fixed monthly bills, outstanding debts, and income sources
- **Savings Goal Tracking**: Progress bar toward your $6,000 savings goal
- **Local Storage**: All data stored locally in your browser

## Setup

1. Open `budget-ai.html` in a web browser
2. To enable AI features, add your Anthropic API key to the JavaScript code (replace the fetch call with proper authorization)

## Usage

- Chat with the AI about purchases: "Spent $50 at grocery store"
- Log income: "Got paid $500 today"
- Track savings: "Put $100 in savings"
- Ask questions: "How am I doing this month?" or "Can I afford this?"

The AI knows your complete financial situation and provides personalized advice.
